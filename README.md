# AsteroidsSpoof
Asteroids Spoof

This is a spoof of Asteroids made in Unity, under the Guidance of Dr. Tim "Dr. T" Chamillard from University of Colorado, Colorado Springs

### Goal

  - Destroy all asteroids

### Controls

  - **Shoot**: *Left Control*
  - **Move Forward**: *Space*
  - **Rotate**: *Right/Left*
